pub async fn set_send_resolution(&self, height: i32)

Set the max resolution that we are currently sending.

Setting this is required for browser clients in the same conference to display the stats that we broadcast.

Note that lib-gst-meet does not encode video (that is the responsibility of your GStreamer pipeline), so this is purely informational.
